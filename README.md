# Ashutosh Kesari Portfolio — Edit Guide

## Folder Structure
```
ashutosh-portfolio/
├── content.js              ← ✅ ONLY FILE YOU NEED TO EDIT
├── index.html              ← Page shell (don't touch)
├── css/
│   ├── main.css            ← Colors, cursor, fonts
│   ├── layout.css          ← Section layouts
│   └── responsive.css      ← Mobile styles
├── js/
│   ├── render.js           ← Builds page from content.js (don't touch)
│   └── main.js             ← Animations & cursor (don't touch)
└── assets/images/
    ├── portrait/           ← Drop your profile photo here
    ├── photography/        ← Drop gallery photos here
    ├── projects/           ← Drop project thumbnails here
    └── design/             ← Drop design/thumbnail images here
```

## To update text/info → Edit content.js
## To add photos → Drop files in assets/images/[folder]/ then update image: paths in content.js

## Deploy: Drag the entire folder to netlify.com/drop → instant live link
